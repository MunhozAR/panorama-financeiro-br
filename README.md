# Panorama Competitivo do Mercado Financeiro Brasileiro

Análise comparativa entre bancos tradicionais, bancos digitais e fintechs no Brasil, utilizando dados públicos do Banco Central e relatórios oficiais.

## 🎯 Pergunta de negócio

Os bancos digitais estão realmente conquistando o mercado financeiro brasileiro, ou os bancos tradicionais ainda mantêm vantagem competitiva em rentabilidade e escala?

## 🏦 Instituições analisadas

**Bancos tradicionais:** Itaú, Bradesco, Banco do Brasil, Santander
**Bancos digitais:** Nubank, C6 Bank, Banco Inter
**Fintechs de pagamento:** PagBank, Mercado Pago, PicPay

## 📊 KPIs avaliados

- Ativo Total e crescimento YoY
- ROE (Return on Equity)
- Carteira de Crédito e Inadimplência
- Índice de Eficiência
- Número de clientes e receita por cliente

## 🛠️ Stack

- **Coleta:** IF.data Bacen, relatórios trimestrais (CVM)
- **Tratamento:** Python (pandas) e Excel
- **Análise:** SQL e Python
- **Visualização:** Power BI
- **Versionamento:** Git/GitHub
